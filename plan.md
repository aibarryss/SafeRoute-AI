# SafeRoute AI — План разработки с ML

## Обзор проекта

**SafeRoute AI** — система построения безопасных маршрутов в Семею с использованием машинного обучения для предсказания уровня опасности районов.

**Трек:** Track 3 — City Safety & Social Services  
**Город:** Семей, Казахстан  
**Хакатон:** SmartScape Hackathon 2026

### Компоненты

1. **ML Pipeline** (`backend/ml/`)
   - `data_generator.py` — генерация синтетических данных для обучения
   - `train_model.py` — обучение модели предсказания опасности
   - `danger_predictor.py` — инференс модель для продакшена
   - `models/danger_model.pkl` — сохранённая обученная модель

2. **Route Planner** (`backend/routes.py`)
   - Построение маршрутов с учётом предсказаний ML
   - Три режима: авто, ребёнок, турист
   - Алгоритм обхода опасных зон

3. **LLM Integration** (`backend/ai.py`)
   - OpenRouter API для генерации объяснений
   - Промпт-инжиниринг на русском языке

## ML Модель

### Задача

**Классификация уровня опасности района** (1-10) на основе признаков:

- Время суток (hour_of_day)
- День недели (day_of_week)
- Тип района (district_type)
- Уровень освещённости (lighting)
- Плотность камер (cctv_density)
- Плотность населения (population_density)
- Исторические инциденты (historical_incidents)
- Тип дороги (road_type)

### Модель

**Алгоритм:** Gradient Boosting (scikit-learn)

**Обоснование выбора:**

- Интерпретируемость (feature importance)
- Быстрый инференс (< 10ms)
- Хорошо работает с табличными данными
- Не требует больших объёмов данных

### Данные

**Источник:** Синтетические данные с обоснованными фичами  
**Объём:** 5000+ примеров  
**Генерация:** Реалистичные распределения на основе реальных городских паттернов
