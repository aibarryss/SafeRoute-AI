## Синтетические данные — Районы Семея

Используем реалистичные данные на основе географии города:

| Район | Уровень опасности | Причина |
|---|---|---|
| Центр | 🟢 3 | Людно, освещено, камеры |
| Железнодорожный | 🟡 5 | Смешанный трафик |
| Промышленный север | 🔴 8 | Мало освещения, редкие прохожие |
| Набережная Иртыша | 🟢 2 | Популярное место, освещено |
| Спальные районы (запад) | 🟡 6 | Тёмные переулки ночью |
| Рынок Шыгыс | 🟡 5 | Карманники, толпа |
| Новостройки (восток) | 🟢 4 | Новая инфраструктура |

## MVP — Минимально рабочий продукт

Если времени мало — фокусируемся только на этом:

1. ✅ Карта Семея с цветными зонами
2. ✅ Один рабочий режим маршрута (например, "Для ребёнка")
3. ✅ ИИ объясняет маршрут текстом
4. ✅ Красивый интерфейс

Остальные режимы можно показать как "в разработке" на презентации.
