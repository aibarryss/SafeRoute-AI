# SafeRoute AI — Архитектура проекта

## Стек технологий

| Слой | Технология | Назначение |
|---|---|---|
| Frontend | HTML + CSS + JavaScript | Интерфейс пользователя |
| Карта | Leaflet.js + OpenStreetMap | Отображение карты Семея |
| Backend | Python + FastAPI | API сервер |
| ИИ | Claude API (claude-sonnet-4-6) | Объяснение маршрутов |
| Данные | JSON (синтетические) | Данные об опасности районов |

## API эндпоинты

### `GET /api/zones`
Возвращает список зон с уровнем опасности для тепловой карты.

**Ответ:**
```json
[
  {
    "id": "center",
    "name": "Центр",
    "danger_level": 3,
    "lat": 50.4111,
    "lng": 80.2275,
    "radius": 800
  }
]
```

### `POST /api/route`
Строит безопасный маршрут и получает объяснение от ИИ.

**Запрос:**
```json
{
  "start": { "lat": 50.4111, "lng": 80.2275 },
  "end": { "lat": 50.4200, "lng": 80.2400 },
  "mode": "child"
}
```

**Ответ:**
```json
{
  "route": [...],
  "danger_score": 2,
  "ai_explanation": "Этот маршрут безопасен для ребёнка, потому что..."
}
```

## Уровни опасности

| Уровень | Цвет | Описание |
|---|---|---|
| 1–3 | 🟢 Зелёный | Безопасно |
| 4–6 | 🟡 Жёлтый | Умеренный риск |
| 7–10 | 🔴 Красный | Опасно |

---

## Режимы маршрутов

| Режим | Логика выбора пути |
|---|---|
| 🚗 На машине | Избегает зоны с авариями и угонами (danger > 6) |
| 👶 Для ребёнка | Только зелёные зоны, людные улицы, без тёмных переулков |
| 🧳 Туристу | Обходит красные зоны, предпочитает центральные улицы |
